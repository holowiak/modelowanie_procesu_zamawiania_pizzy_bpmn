# Modelowanie Procesu Zamawiania Pizzy (BPMN)

## Cel projektu

Projekt został stworzony jako przykład do portfolio w celu zaprezentowania:
- podstawowego modelowania procesów w notacji BPMN
- biznesowego podejścia do myślenia procesowego
- czytelnej i logicznej wizualizacji procesu end-to-end
- umiejętności wizualnego przedstawiania procesów biznesowych w formie diagramu

## Opis projektu

Projekt przedstawia uproszczony proces zamawiania pizzy, wykonany w draw.io.  
Model koncentruje się na:
- pełnym przepływie procesu od złożenia zamówienia do jego realizacji
- jasno zdefiniowanych rolach i odpowiedzialnościach
- kluczowych punktach decyzyjnych (dostawa / odbiór osobisty, płatność gotówką / kartą)
- podstawowej obsłudze wyjątków
- spójności i czytelności procesu z perspektywy biznesowej

## Zakres procesu

Proces obejmuje:
- inicjację zamówienia przez klienta
- przyjęcie i realizację zamówienia przez personel restauracji
- dostawę zamówienia lub odbiór osobisty
- realizację płatności i zakończenie procesu

## Zawartość repozytorium

Repozytorium zawiera:
- diagram procesu biznesowego wykonany w notacji BPMN
- plik PDF wygenerowany z draw.io, prezentujący kompletny przepływ procesu
